
# 📊 Student Marks Analysis

This is a simple Python project to analyze student marks from a CSV file.

## 🚀 Features
- Calculates average marks of each student
- Identifies the topper
- Computes class average

## 🛠️ Technologies Used
- Python
- CSV File Handling

## 📂 Project Structure
- marks.csv → dataset
- main.py → Python script

## ▶️ How to Run
1. Download the files
2. Run main.py using Python
